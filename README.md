## ATM Simulation System

This project is a simple ATM Simulation System developed using Java. It is a console-based application that allows users to perform basic banking operations after entering the correct PIN.

## Features

- PIN verification
- Balance inquiry
- Cash deposit
- Cash withdrawal
- Mini statement
- Change PIN
- View account details
- Exit the application

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## Files Included

- Main.java
- ATM.java
- User.java
- BankAccount.java
- Transaction.java

## How to Run

1. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, or VS Code).
2. Compile all the Java files.
3. Run the `Main.java` file.
4. Enter the correct PIN to access the ATM menu.

## Learning Outcome

This project helped me understand Java programming concepts such as classes, objects, encapsulation, constructors, methods, and menu-driven applications. It also improved my understanding of basic banking operations and transaction handling.
